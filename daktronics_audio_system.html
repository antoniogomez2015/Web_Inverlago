​Sportsound 150 Indoor Sound System

Sistema de Sonido Deportivo 150, para Interiores

Muestra bajo techo

Información general

Este sistema de sonido estándar de interior Sportsound® está especialmente diseñado para los pequeños gimnasios de interior. El sistema de un solo punto está permanentemente montado en la pared y proporciona una cobertura uniforme y niveles capaces de superar ruido de la multitud. Gráficos o anuncios personalizados se pueden imprimir en la malla frontal envolvente.

​Sportsound 500HD Sound System

Sistema de Sonido Deportivo 500HD, para Exteriores

Muestra en exteriores

Información general

Sonido diseñado para proporcionar alto impacto, refuerzo de sonido para una variedad de instalaciones, incluyendo fútbol, lacrosse, softbol y atletismo. El sistema se instala en una posición permanente y proporciona un sonido de rango completo y una cobertura uniforme a niveles capaces de superar ruido de la multitud. El 500HD, está diseñado para instalaciones deportivas más pequeñas; es fácil de instalar y de operar. Consola de Locutor, ofrece una interfaz simplificada para el sistema de sonido, que ofrece uno de los reproductores MP3, y dos entradas de micrófono.

​Sportsound 1500HD Sound System

Sistema de Sonido Deportivo 1500HD, para Exteriores

Muestra en exteriores

Información general

El sistema de sonido Sportsound 1500HD, es similar al sistema 500HD con un solo mueble de diseño que se monta en un solo lugar. Sin embargo, el 1500HD, tiene mayores capacidades que el 500HD, con un patrón de cobertura más amplia y una mayor calificación de decibelios, lo que aumenta considerablemente el rendimiento y el tamaño típico del estadio que este soporta.
