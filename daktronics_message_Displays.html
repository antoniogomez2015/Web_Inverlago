​Tri-Color Message Displays

Pantallas a Tres (3), Colores para Mensajes

Muestra bajo techo

Información general

Pantallas para Exteriores a tres colores Matrix, completa pantalla de mensajes. El tricolor AF-6200, ofrece up-to-the-minute horarios, información sobre el estacionamiento, las direcciones y la publicidad, guía para vehículos dentro de las instalaciones de estacionamiento. Pantalla increíblemente resistente orienta a los clientes con el espaciado de píxeles 8 mm, legible para el texto claro que se puede ver desde un campo ancho y largo de vista. Potencia industrial NEMA-4X, el AF-6200, a prueba de agresiones, permite mantener fácilmente en lugares donde se requiere durabilidad.
